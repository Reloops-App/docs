# Notifications and activity overview

Notifications help you see review activity without opening every project manually.

!\[Notifications panel showing review activity and file updates\](/files/d3923493e776e183dbb1218e34ea73a1a91c2788)

Use notifications to track:

\* comments and mentions
\* review activity
\* file updates that need attention

Use this page when you need the high-level purpose of notifications.

For tab-by-tab detail, open \[Notifications panel\](/reloops-docs/workspace-management/notifications-panel.md).
