Notifications help you see review activity without opening every project manually.

![](https://reloops.gitbook.io/reloops-docs/~gitbook/image?url=https%3A%2F%2Fcontent.gitbook.com%2Fcontent%2FqpxgNvXAcPEWnIZeWuLE%2Fblobs%2F57oGhFGA9uIDF9SBB4Hs%2Fnotifications%2520panel%2520all%2520tab.png&width=768&dpr=3&quality=100&sign=5ac8c9ed&sv=2)

Notifications panel showing review activity and file updates

Use notifications to track:

- comments and mentions
 
- review activity
 
- file updates that need attention
 

Use this page when you need the high-level purpose of notifications.

For tab-by-tab detail, open [Notifications panel](https://reloops.gitbook.io/reloops-docs/workspace-management/notifications-panel).
