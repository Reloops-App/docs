Notifications panel showing review activity and file updates

### 

How to use the tabs

- **All** shows the full activity feed.
 
- **Mentions** helps you catch direct feedback quickly.
 
- **Reviews** focuses on review actions and decisions.
 
- **Files** highlights asset-level updates.
 

Check the panel when you need to know what changed recently or whether someone left feedback that needs a response.
