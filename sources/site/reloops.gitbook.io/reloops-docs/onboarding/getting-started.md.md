# Getting started

## Getting started with Reloops

Use this guide for the first working setup.

By the end, you will have a workspace, a project and one asset ready for review.

For the screen map, open \[Product walkthrough\](/reloops-docs/onboarding/product-walkthrough.md).

{% stepper %}
{% step %}

## Step 1: Create or join a workspace

Sign in, then open an existing workspace or create a new one.

Choose a clear name if you create the workspace.

Use a client, brand or team name when possible.
{% endstep %}

{% step %}

## Step 2: Create your first project

Inside the workspace, create a project for the work you want to review.

To create it:

1. Open the Projects page
2. Select \*\*New Project\*\*
3. Enter a project name
4. Confirm to create it

Start with an empty project unless you already need a template.
{% endstep %}

{% step %}

## Step 3: Upload your first asset

Select the \*\*Upload\*\* button and choose files from your device or drag and drop them into the upload area.

After upload, the file appears in the Files view as its own asset.
{% endstep %}

{% step %}

## Step 4: Review and collaborate

Open any asset to start reviewing it.

You can:

\* leave comments
\* add annotations
\* assign the asset
\* update its status
\* share it with a reviewer
 {% endstep %}

{% step %}

## Step 5: Track progress

As work moves forward, update statuses such as \*\*Needs review\*\*, \*\*In Review\*\* or \*\*Approved\*\*.

Use Kanban when you want to track progress across the whole project.
{% endstep %}
{% endstepper %}

## What to do next
