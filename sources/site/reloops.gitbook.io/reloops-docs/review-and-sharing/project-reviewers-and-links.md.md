# Project reviewers and links

### Project reviewers and links

Project review ownership is visible from the Reviewers and Links tabs.

App route: \`/workspace/:workspaceId/projects/:projectId\`

!\[Project reviewers tab showing a guest reviewer and assigned asset\](/files/9370b5950f106fd41a23a8a3070bec5749185ae7)

Use the Links tab to check which asset links are active and whether they expire.

!\[Project links tab showing an active asset share link\](/files/9353da8b890d569167600ee5c756aad9e2b6d17e)

### When to check these tabs

\* A reviewer says they cannot access an asset.
\* A project has several external stakeholders.
\* A review window is finished and old links should be cleaned up.
