### 

Upload assets in Reloops

Uploading assets is how work begins inside a project. Once uploaded, each file becomes an asset that you can review, comment on and track through its workflow.

You can upload assets at any time from inside a project.

---

### 

Where you can upload assets

Assets can only be uploaded inside a project.

To upload:

1. Open a workspace
 
2. Enter a project
 
3. Select the **Upload** button in the top right corner
 

This opens the upload window.

---

### 

How to upload assets

Reloops offers two simple ways to upload files.

#### 

Drag and drop

You can drag files from your computer and drop them into the upload area. The upload starts automatically.

#### 

Browse files

You can select **Browse files** and choose files from your device.

Both methods support uploading multiple files at once.

---

### 

Supported file formats

Reloops supports common image and video formats.

#### 

Images

- JPG
 
- PNG
 
- GIF
 
- WEBP
 
- SVG
 

#### 

Videos

- MP4
 
- MOV
 
- WEBM
 
- MKV
 

Large files upload smoothly using parallel multipart uploads.

---

### 

What happens after upload

After upload:

- each file becomes an asset
 
- assets appear in the Files view
 
- assets can be searched, filtered and sorted
 
- assets are ready for review
 

By default, newly uploaded assets start with **No status**.

---

### 

Uploading multiple assets

You can upload multiple files at the same time. Each file is processed individually and appears as its own asset inside the project.

This is useful when uploading batches of work.

---

### 

Common upload tips

- Make sure you are inside the correct project before uploading
 
- Use clear file names to make assets easier to find
 
- Upload related files together to keep reviews organized
 

---

### 

What to do next

After uploading your assets, you can start reviewing them.

Next, open [Project assets view](https://reloops.gitbook.io/reloops-docs/projects-and-assets/project-assets-view) to manage the uploaded files.

Then use [Asset review](https://reloops.gitbook.io/reloops-docs/review-and-sharing/asset-review) to start feedback.
