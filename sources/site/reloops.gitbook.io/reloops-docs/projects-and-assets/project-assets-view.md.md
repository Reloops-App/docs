# Project assets view

The Assets view is the main file workspace inside a project.

Use it to browse assets, start uploads, open files for review and manage work in one place.

App route: \`/workspace/:workspaceId/projects/:projectId\`

!\[Project assets page with the Upload menu open\](/files/73e2f664c46955ef84c40ca97b03fd6cb3be6439)

## What you can do from the Assets view

\* Start file or folder uploads.
\* Open any asset for review.
\* Browse assets as cards in the Files view.
\* Share assets and track review progress.

## Upload options

\* \*\*Upload Files\*\* adds one or more selected files.
\* \*\*Upload Folder\*\* keeps a folder-based upload grouped together.

After upload, each file becomes an asset card.

Open an asset card to review it, comment on it, assign it, update status or share it.

For the full upload flow, continue with \[Upload assets\](/reloops-docs/projects-and-assets/upload-assets.md).
