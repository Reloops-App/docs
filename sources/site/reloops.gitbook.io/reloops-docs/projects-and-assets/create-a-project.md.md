# Create a project

Projects keep the assets, folders, reviewers and share links for a campaign or review round together.

App route: \`/workspace/:workspaceId/projects\`

!\[Projects page showing project cards and the New Project button\](/files/03f9feea8692a2e9a2c49802a2a2a8e65eeaddf5)

## Create a project

1. Open the correct workspace.
2. Go to \*\*Projects\*\*.
3. Select \*\*New Project\*\*.
4. Choose a template or start with no template.
5. Create the project.

The Social Media template creates a starting folder structure for campaign planning, source files and platform deliverables.

!\[Create New Project modal with the Social Media template selected\](/files/b495e157d2e9c6f12d01f661d9d7443da5fecad8)
