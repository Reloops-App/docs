A project is the central container for a campaign or review round in Reloops. It holds your assets, folders, reviewers, and share links together so everyone on the team works from the same place. You can create as many projects as you need inside a workspace, and each one stays completely separate from the others.

## Create a project

Navigate to your workspace and follow these steps to create a new project.

**App route:** `/workspace/:workspaceId/projects`

<Steps>
  <Step title="Open the correct workspace">
    Select the workspace you want to create the project in from the workspace switcher.
  </Step>
  <Step title="Go to Projects">
    Click **Projects** in the left-hand navigation to open the Projects page.
  </Step>
  <Step title="Select New Project">
    Click the **New Project** button to open the Create New Project modal.
  </Step>
  <Step title="Choose a template or start empty">
    Select a template to get a pre-built folder structure, or leave the template unselected to start with an empty project.
  </Step>
  <Step title="Confirm and create">
    Name your project and click **Create** to finish. Your new project opens immediately and is ready for uploads.
  </Step>
</Steps>

## The Social Media template

When you select the **Social Media** template, Reloops creates a starting folder structure designed for campaign planning. The template includes folders for source files, campaign planning materials, and platform-specific deliverables, so your team can start uploading work straight away without having to set up the structure manually.

Choose this template when you are managing a campaign that produces assets across multiple channels or output formats.
